# Emergency Service Directory

A simple web application to provide quick access to important emergency numbers in Bangladesh.

## Project Overview

This project aims to offer a clean, responsive interface where users can view and call various emergency services (police, fire, ambulance, helplines, etc.).  
It is built with HTML, CSS (Tailwind CSS), and JavaScript (DOM manipulation).

## Live-link

You can view the live version here: https://mohyminulislam-mi.github.io/Emergency-Service-Call/

## Technologies Used

- HTML5
- CSS3
- Tailwind CSS
- JavaScript
- DOM

## Features

- Lists major emergency service numbers in Bangladesh (e.g., police, fire service, ambulance, anti-corruption helpline, electricity helpline)
- “Call” and “Copy” buttons for each number (for easier access)
- Responsive layout: adapts for mobile and desktop screens
- Simple, clean UI for fast usage in urgent situations
